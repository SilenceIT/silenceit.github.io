基于Vite项目笔记

## 运行程序
  1. 安装依赖
shell
# npm
npm i --registry=https://registry.npmmirror.com

# pnpm
pnpm i --registry=https://registry.npmmirror.com
  1. 运行程序
shell
# 开发环境
npm run docs:dev

# 构建静态文件
npm run docs:build

# 线上环境环境
npm run docs:preview

# 云端部署笔记
npm run docs:deploy

工程结构解析

├── .vitepress
│   ├── dist                   # 打包文件夹（可删除重新打包）
│   ├── theme                   # 全局主题样式
│   ├── config.ts              # 配置文件
├── view                       # 笔记文档
├── .prettierrc.json           # prettier 配置
├── .gitignore                 # git 忽略文件
├── index.md                   # 首页
├── package.json               # package.json 依赖
